# myCordovaPlugin
Template for customizing cordova plugin in ionic2

Making Cordova Plugin from scratch

Create plugin skeleton
```
cd plugins
plugman create --name myPlugin --plugin_id cordova-plugin-my --plugin_version 0.0.1
```
